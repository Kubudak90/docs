Returns information about a block by its block number.

## Parameters

<ParamField body="blockParameter" type="string" required>
  Block number in hexadecimal, or a block tag: `latest`, `earliest`, `pending`, `safe`, or `finalized`.
</ParamField>

<ParamField body="fullTransactions" type="boolean" required>
  If `true`, returns full transaction objects in the `transactions` array. If `false`, returns only transaction hashes.
</ParamField>

## Returns

<ResponseField name="result" type="object | null">
  A block object, or `null` if no block was found.

  <Expandable title="Block object fields">
    <ResponseField name="number" type="string">
      Block number as a hex string. `null` when the block is pending.
    </ResponseField>
    <ResponseField name="hash" type="string">
      32-byte hash of the block. `null` when the block is pending.
    </ResponseField>
    <ResponseField name="parentHash" type="string">
      32-byte hash of the parent block.
    </ResponseField>
    <ResponseField name="nonce" type="string">
      8-byte proof-of-work nonce. Always `"0x0000000000000000"` on Base (proof-of-stake).
    </ResponseField>
    <ResponseField name="sha3Uncles" type="string">
      32-byte SHA3 hash of the uncles data in the block. Always the empty-uncle-list hash on Base.
    </ResponseField>
    <ResponseField name="logsBloom" type="string">
      256-byte bloom filter for the logs of the block.
    </ResponseField>
    <ResponseField name="transactionsRoot" type="string">
      32-byte root of the transaction trie.
    </ResponseField>
    <ResponseField name="stateRoot" type="string">
      32-byte root of the final state trie.
    </ResponseField>
    <ResponseField name="receiptsRoot" type="string">
      32-byte root of the receipts trie.
    </ResponseField>
    <ResponseField name="miner" type="string">
      20-byte address of the fee recipient (block proposer on Base).
    </ResponseField>
    <ResponseField name="difficulty" type="string">
      Block difficulty. Always `"0x0"` on Base (proof-of-stake).
    </ResponseField>
    <ResponseField name="totalDifficulty" type="string">
      Total chain difficulty. Always `"0x0"` on Base (proof-of-stake).
    </ResponseField>
    <ResponseField name="extraData" type="string">
      Hex-encoded extra data field of the block.
    </ResponseField>
    <ResponseField name="size" type="string">
      Integer size of the block in bytes as a hexadecimal.
    </ResponseField>
    <ResponseField name="gasLimit" type="string">
      Maximum gas allowed in this block.
    </ResponseField>
    <ResponseField name="gasUsed" type="string">
      Total gas used by all transactions in this block.
    </ResponseField>
    <ResponseField name="timestamp" type="string">
      Unix timestamp of when the block was collated.
    </ResponseField>
    <ResponseField name="transactions" type="array">
      Array of transaction hashes (when `fullTransactions` is `false`) or full transaction objects (when `true`).
    </ResponseField>
    <ResponseField name="uncles" type="array">
      Array of uncle block hashes. Always `[]` on Base.
    </ResponseField>
    <ResponseField name="baseFeePerGas" type="string">
      Base fee per gas in this block (EIP-1559).
    </ResponseField>
    <ResponseField name="withdrawals" type="array">
      Array of validator withdrawals included in the block (EIP-4895).
    </ResponseField>
    <ResponseField name="withdrawalsRoot" type="string">
      32-byte root of the withdrawals trie (EIP-4895).
    </ResponseField>
  </Expandable>
</ResponseField>

## Flashblocks Behavior

On a [Flashblocks-aware endpoint](/base-chain/flashblocks/apps#rpc-endpoints), use `"pending"` as the block parameter to query preconfirmed state — data available up to 200ms before the next full block is sealed.

When called with `"pending"`, returns the current Flashblock — the block actively being built with preconfirmed transactions, sealed approximately every 200ms.

<CodeGroup>
```bash cURL (preconfirmed state)
curl https://mainnet-preconf.base.org \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_getBlockByNumber","params":["pending",false],"id":1}'
```
</CodeGroup>
