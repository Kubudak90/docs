Returns transaction information for a given block hash and transaction index position.

## Parameters

<ParamField body="blockHash" type="string" required>
  The 32-byte hash of the block.
</ParamField>

<ParamField body="transactionIndex" type="string" required>
  The transaction's index position in the block as a hexadecimal integer (e.g., `"0x0"` for the first transaction).
</ParamField>

## Returns

<ResponseField name="result" type="object | null">
  A transaction object with the same fields as [`eth_getTransactionByHash`](/base-chain/reference/rpc-methods/eth_getTransactionByHash), or `null` if no transaction was found at that position.
</ResponseField>

## Example

<CodeGroup>
```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionByBlockHashAndIndex",
  "params": [
    "0x3a4e8c5d7f2b1a6e9d0c4f8b3e7a2d5c8f1b4e7a0d3c6f9b2e5a8d1c4f7b0e3",
    "0x0"
  ],
  "id": 1
}
```

```json Response
{

    "from": "0xd3CdA913deB6f4967b2Ef66ae97DE114a83bcc01",
    "to": "0x4200000000000000000000000000000000000006",
    "value": "0x2c68af0bb14000",
    "type": "0x2"
  }
}
```
</CodeGroup>
